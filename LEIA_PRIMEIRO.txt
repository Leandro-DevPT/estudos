┌──────────────────────────────────────────────────────────────────────────────┐
│ ❓ TROUBLESHOOTING (Se Algo Der Errado):                                     │
└──────────────────────────────────────────────────────────────────────────────┘

❌ "npm não é reconhecido"
   → Node.js não está instalado ou não está no PATH
   → Solução: Reinstale Node.js e reinicie o PC

❌ "Erro: EADDRINUSE, port 3000 already in use"
   → Outra aplicação está usando a porta 3000
   → Solução: Mude a porta em server.js ou feche o programa

❌ "npm ERR! code ERESOLVE"
   → Problema com dependências
   → Solução: npm install --force

❌ "Exercício não aparece"
   → Servidor não foi reiniciado
   → Solução: Ctrl+C e npm start novamente

❌ "Página não carrega (http://localhost:3000)"
   → Servidor não está rodando
   → Solução: Execute npm start no terminal
