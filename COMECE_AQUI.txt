❓ TROUBLESHOOTING:

"npm não encontrado"
→ Reinstale Node.js

"Porta 3000 em uso"
→ Mude const PORT em server.js

"Erro ao instalar"
→ Execute: npm install --force

"Exercício não aparece"
→ Reinicie o servidor (Ctrl+C e npm start)

🔗 API REST ENDPOINTS:

Autenticação:
  POST   /api/register         Registrar novo usuário
  POST   /api/login            Fazer login

Exercícios:
  GET    /api/exercises        Listar exercícios
  GET    /api/exercises/:id    Detalhes do exercício
  POST   /api/submit-exercise  Enviar resposta

Usuário:
  GET    /api/user/:id         Dados do usuário
  GET    /api/user/:id/progress Progresso do usuário
  GET    /api/stats/:id        Estatísticas

Ranking:
  GET    /api/ranking          Top 10 usuários
